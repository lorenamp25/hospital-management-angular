# Hospital Management System (Angular)

Live Demo:
https://hospital-management-angular-l4vy57shl-lorenas-projects-17e1e0a4.vercel.app/

Hospital management application built with Angular to manage patients, doctors and appointments.

## Features
- Patient management
- Doctor management
- Appointment scheduling
- Simple dashboard interface
- Responsive design

## Technologies
- Angular
- TypeScript
- HTML / CSS

## Installation

```bash
npm install
ng serve
```
